# RESEARCH-BASED-CANCER-DETECTION-WITH-EXPLAINABLE-AI

In this project, I bridge the gap between academic research and real-world
Medical AI. I move away from using pre-packaged, beginner-friendly models
and instead take on the challenge of a Senior AI Researcher: implementing a novel,
state-of-the-art architecture from a research paper.

I build ThyroCheck AI, a diagnostic tool that classifies thyroid nodules from
complex ultrasound images. To achieve high precision, I code a custom deep
learning architecture known as FibonacciNet from scratch.

Crucially, because medical professionals cannot trust a "black box" algorithm, I
integrate Explainable AI (XAI). I implement Grad-CAM to generate visual
heatmaps, proving exactly why the model made its diagnosis. Finally, I package
this intelligence into an automated pipeline that generates comprehensive, clinician-
ready diagnostic reports.
